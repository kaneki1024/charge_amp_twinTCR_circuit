# 卒論にて作成した圧電センサ用の回路
力を加えると電荷、電圧が発生する圧電体という材料がある。中心対称性が無ければ圧電性は示すが、圧電体は誘電体であるため出力インピーダンスが非常に大きいという特徴をもつ。よって圧電体の電圧を計測ではなく表面の電荷計測を行い、センサとして利用する。
